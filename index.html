About the PEI Dashboard

The Pedestrian Environment Index (PEI) is a nationwide metric designed to evaluate the walkability and safety of pedestrian infrastructure across the United States.

This interactive dashboard visualizes eight core built environment indices:

  • Commercial Density & Intersection Density
  • Land Use Diversity & Population Density
  • Road Safety (RSI)
  • Green Space & Bike Infrastructure
  • Public Transit Access (PTAL)

Use the swipe tools and layer selectors to track structural changes across 2013, 2017, and 2022.

More features are to come, including demographic data overlays and additional timeline years!

For more information on how each index was calculated, please visit: https://vip-smur.github.io/25fa-mobility/
